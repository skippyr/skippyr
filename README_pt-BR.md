## ❡ Sobre
Um engenheiro de software ambicioso dedicado a seguir uma filosofia focada no desenvolvimento de softwares de múltiplos domínios com qualidade de elite, destacando-se por serem ajustados ao baixo nível e com perfeição nos detalhes, únicos e projetados para serem intuitivos tanto para usuários quanto para desenvolvedores, mirando todas as principais plataformas do mercado.

Além de qualquer lógica possível, tenho um apreço inesperado por My Little Pony, pelo simbolismo das culturas tribais e pelo espírito ardente herdado dos dragões. Esses elementos me ajudam a criar mundos mais envolventes onde meu software pode habitar.
