## ❡ Sobre
Um engenheiro de software que gosta de construir tudo que é possível: de sistemas empresariais e aplicações de desktop a circuitos embarcados e sistemas operacionais. Meu objetivo é sempre de liderar projetos para as melhores soluções possíveis, simplificando qualquer complexidade para que você possa pular passos desnecessários—por isso meu apelido é "skippyr" (uma palavra-valise inglesa de "capitão" e "pular").

Fora da lógica dos códigos, eu possuo uma paixão inesperada por My Little Pony, cultura tribal e dragões, elementos que servem para adicionar um pouco de mágica em cada dia de minha vida, ajudando-me a ter diversão e obter ideias criativas para os meus projetos.
